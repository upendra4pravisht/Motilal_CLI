# Motilal_CLI
Motilal Oswal CLI version to place trade
In order to trade with CLI, there are two excel files which are as follows...
1- Client Login Credentis 
   client_code	
   apikey	
   totp	
   dob	
   password	
   broker	
   group	
   multiplier	
   dealer_id

2- Trade Detail
   Client ID	
   Order Type	
   Symbol	
   Exchange	
   Quantity	
   Price/Limit 
   Price	
   Is Intraday	
   SL 
   Trigger Price
